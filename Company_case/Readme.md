Library Management System
This project is a technical assessment developed for a job interview. Its primary purpose is to demonstrate my core skills in C# and ASP.NET Core through a set of functional features and a structured architectural approach.

Overview
The application is designed to showcase clean code practices and modular design, featuring a 3-tier architecture to effectively separate concerns between the UI, business logic, and data storage.

Key Features
Library Catalog: A functional web interface to browse and search for books.

Data Storage: Initially implemented using XML, with a design that allows for easy migration to a SQL database.

Functional Utilities: Includes custom services for mathematical calculations, such as Fibonacci sequences and specific date logic (e.g., Easter calculation).

Code Quality: The project includes a dedicated test suite using XUnit to ensure reliability and demonstrate the importance of unit testing.

Technical Stack
Language: C#

Framework: ASP.NET Core MVC

Testing: XUnit

Data Format: XML

Getting Started
Clone the repository.

Open the solution in your IDE (e.g., Visual Studio, Rider, or VS Code).

Ensure the .NET SDK is installed.

Run the project using dotnet run or through your IDE's interface.
